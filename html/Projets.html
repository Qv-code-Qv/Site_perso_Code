Site Web pour une Mairie

    • Développement d'un site fictif sous Symfony

  • Ce projet a été réalisé dans le cadre de mon BTS SIO SLAM. Il consistait à réaliser un site Web pour une entreprise fictive nomé "RetroManGames". Ce site permet de pouvoir consulter les différentes salles d'arcade, de s'inscrire et se connecter sur sa page profil utilisateur. Ce projet a été réalisé entièrement sous Symfony 3 avec les différents outils associés comme fosuserbundle pour la gestion des inscriptions et des connexions. Pour pouvoir accèder au site RetroManGames, un serveur web a été configuré avec un Raspberry Pi sous raspbian. Site RetroManGames
